New Accession Data Entry - User Notes

  • Codeno:
    • next available autogenerated (previous +1), change as needed
    • First accession of new year needs to be manually changed to YYYY00001 in entry field!
  • Genus:
    • must be valid genus name ONLY
    • Typically check
      1. World Checklist of Selected Plant Families - not all families in database, but most up to date for those that are
      2. World Flora Online
      3. PlantList new data is no longer be added here (static since 2013)
    • Source citation is generally first entry into the credits field, usually first thing I do after creating a new accession.
    • Genus must appear in the bcm.classify table for proper taxonomic lookup
  • Species:
    • can include varietal names
    • some punctuation, quote marks etc can cause data errors, best to exclude special characters
    • FUTURE: error checking & correction code could be implemented
  • Source:
    • Simple descriptor
    • Detailed provenance data can be added later
  • Location:
    • initial bench location - can be changed later
  • Quantity:
  • Project:
    • GEN_COLL is default
    • ALL CAPS is standard
